Detailed Description

Class representing a list of IDs pointing to instances that are related by some conventional criterion that is referred as the "symmetry".

Note that the actual criterion that defined a "symmetry" relation is not defined here. So, this list does not know what is the criterion that puts its entries in relation with each other. Moreover, the list does not know what the entries are, just their numerical identifier.
